# CapSimApi
Python utility to retrieve CapSim data

## Installation
```
git clone https://github.com/nicholasrobinson/CapSimApi.git
cd CapSimApi
pip install requirements.txt
```

## Setup
```
cp config.ini.sample.ini config.ini
```
Add:
- sim_id
- student_key
- username
- password

## Usage
```
$ python CapsimApi.py
```
